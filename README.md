# -Application-Web-de-gestion-des-documents-et-de-courri-res
Application web de gestion des documents et courriers pour l'AREF de Rabat-Salé-Kénitra. Permet l'archivage, la consultation, le suivi et la recherche des documents, avec un dictionnaire scientifique. Développée en PHP, MySQL, HTML/CSS, et VS Code.
